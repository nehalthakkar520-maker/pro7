# Multi Utility Toolkit

## Project Overview
Multi Utility Toolkit is a Python-based application that demonstrates the use of built-in modules such as datetime, math, random, and uuid. The project provides multiple utility functions through a menu-driven interface.

## Features

### 1. Date and Time Module
- Display current date and time.

### 2. Math Module
- Calculate square root.
- Calculate factorial.

### 3. Random Module
- Generate random numbers.
- Generate OTP.

### 4. UUID Module
- Generate unique identifiers.

### 5. File Handling Module
- Write data to a file.
- Read data from a file.

### 6. Dynamic Module Exploration
- Display module attributes using dir() function.

## Modules Used
- datetime
- math
- random
- uuid

## Functions Included
- current_datetime()
- math_operation()
- random_number()
- generate_otp()
- generate_uuid()
- write_file()
- read_file()
- module_info()

## How to Run

1. Open terminal or command prompt.
2. Navigate to project folder.
3. Run the program:

```bash
python main.py
```
